Was ist jQuery?

jQuery ist eine schnelle, kleine und funktionsreiche JavaScript-Bibliothek. Es macht Dinge wie das Durchlaufen und Bearbeiten von HTML-Dokumenten, Ereignisbehandlung, Animation und Ajax viel einfacher mit einer benutzerfreundlichen API, die in einer Vielzahl von Browsern funktioniert. Mit einer Kombination aus Vielseitigkeit und Erweiterbarkeit hat jQuery die Art und Weise verändert, wie Millionen von Menschen JavaScript schreiben.
Die jQuery-Bibliothek enthält die folgenden Funktionen:

- HTML/DOM-Bearbeitung
- CSS-Bearbeitung
- HTML-Ereignis-Methoden
- Effekte und Animationen
- AJAX
- Utilities

Tipp: Für jQuery gibt es ausserdem Plugins für fast jede Aufgabe, die es gibt.

Warum jQuery?

Es gibt viele andere JavaScript-Bibliotheken, aber jQuery ist wahrscheinlich die beliebteste und auch die am besten erweiterbare. Viele der größten Unternehmen im Web verwenden jQuery, wie z. B. Google, Microsoft, IBM oder Netflix.

Canvas

Das HTML canvas-Element wird verwendet, um Grafiken im Handumdrehen über JavaScript zu zeichnen.
Das canvas-Element ist ein Container für Grafiken. JavaScript wird verwenden, um dann die Grafiken tatsächlich zu zeichnen.
Canvas verfügt über mehrere Methoden zum Zeichnen von Pfaden, Kästchen, Kreisen, Text und zum Hinzufügen von Bildern.
